# 83Plus:BCALLs:4ED9

Source: https://wikiti.brandonw.net/index.php?title=83Plus:BCALLs:4ED9

## Synopsis

**Unofficial Name:** ReceiveAck

**BCALL Address:** 4ED9

"Receives 4-byte acknowledgement packet over the link port."

## Inputs

None.

## Outputs

None.

## Destroys

All registers.

## Comments

"Will throw ERR:LINK if any problems are found, including receiving a
different packet."
